#  

## Main Goal 

Create a functional web tool for the collection of mockup emergency room clinical records from medical doctors to build a corpus of texts.

## Possible Secondary Goals

* Use in a broader reach and let different medical professionals add records while stating affiliation and experence.

* Collect personal descriptions of symptoms from people 

## Method

* Use limeSurvey
* Create new format

## Design 

### Required features

* Statement of professional experience
* Separation of anamnesis sections
* Simple user tagging for symptoms and findings
* Input validation


### Optional features

* Account
* User feedback
* Text highlighting when marking symptom/findings
